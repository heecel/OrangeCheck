# OrangeCheck
Предметная область проекта — система учёта посещаемости студентов в образовательных учреждениях. Разрабатываемое веб-приложение предназначено для автоматизации процесса учёта посещаемости студентов. Система позволит преподавателям быстро отмечать присутствие студентов на занятиях, хранить данные в электронной базе и формировать отчёты.
